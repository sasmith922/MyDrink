# DrakeMaye Frontend

Expo + React Native + TypeScript app for the DrakeMaye MVP.

## Run

```bash
cd frontend
npm install
npx expo start
```

Set backend URL if needed:

```bash
EXPO_PUBLIC_API_BASE_URL=http://localhost:8080 npx expo start
```

## App Pages

- Feed
- Snap (placeholder + manual logging)
- Stats
- Catalogue
- Profile

## Intentionally Not Implemented

- AI image recognition
- Image upload
- Computer vision
- Authentication
- Production deployment
